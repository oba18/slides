前回の復習

>>>

1から10000まで足し合わせたら？？

>>>

```c
int main(void){

  int total = 0;
  for(int i=0;i<=10000;i++){
    total = total + i;
  }
  printf("%d",total);

  return 0;
}
```

>>>


for文の書き方  

```c
for (初期値;繰り返しを続けるか判定;増え方){
  処理;
}
```

>>>

初期値
```c
int i=0
```
`i`を使って処理を回す  
最初は定義なので，型宣言が必要

>>>

繰り返しを続けるか
```c
i<=10000
```
この部分がTrueなら処理を続ける  
例えばiが100なら10000以下なのでTrue  
そのため処理(ループ)を続ける

>>>

増え方
```c
i++
```
処理(ループ)を行ったときに，初期値として設定した`i`をどのように増やすか(処理をするか)  
```c
i++
i=i+1
i+=1
```
これらは全部同じ意味で，`i`に1足して`i`を更新するという意味

プログラムの流れを追っていきましょう

>>>

`i`の値を出力する
```c

  for(int i=1; i<=10; i++){
    for(int j=1; j<=3; j++){
      printf("%d",i); // iの値を出力する
    }
  }
  return 0;
}
```

>>>

```c
int main(void){
  for(int i=1; i<=10; i++){

    // ----- 3回繰り返す -----
    for(int j=1; j<=3; j++){
      printf("%d",i); // iの値を出力する
    }
    // -----------------------

  }
  return 0;
}
```
